# ポケカジャッジ道場

ポケモンカードゲームのジャッジ資格取得のための勉強用Webアプリケーションです。

## 特徴

- 📚 **5つのカテゴリー**: 基本ルール、ペナルティ、大会運営、ゲームメカニクス、シナリオ問題
- 🎯 **クイズ形式**: 4択問題で楽しく学習
- 📊 **進捗管理**: 正答率やカテゴリー別の成績を記録
- 💾 **ローカルストレージ**: ブラウザに進捗を自動保存
- 🎨 **モダンUI**: TailwindCSSによる美しいデザイン

## 技術スタック

- **フレームワーク**: Next.js 14 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **状態管理**: Zustand
- **データ保存**: JSON + localStorage
- **アイコン**: lucide-react

## セットアップ

### 必要要件

- Node.js 18以上
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

```
poke_judge_academy/

│   ├── types.ts          # 型定義
```

## 使い方

1. **ホームページ**: カテゴリーから学習したい分野を選択
2. **クイズ選択**: カテゴリーと問題数を設定
3. **クイズ実行**: 4択から正解を選択し、解説を確認
4. **結果確認**: スコアとランクを表示
5. **進捗管理**: 学習履歴と統計を確認

## カテゴリー

### 基本ルール
ゲームの基本ルール、ターン構造、勝利条件など

### ペナルティ
警告、ゲームロス、失格の基準と対応

### 大会運営
スイスドロー、ラウンド管理、デッキリストなど

### ゲームメカニクス
カード効果の処理、タイミング、優先順位など

### シナリオ問題
複雑な状況での裁定、実践的なジャッジング

## データ管理

### 問題データの追加

`data/questions/` ディレクトリ内のJSONファイルに問題を追加できます。

```json
{
  "id": "rules-001",
  "category": "rules",
  "difficulty": "beginner",
  "question": "問題文",
  "options": ["選択肢1", "選択肢2", "選択肢3", "選択肢4"],
  "correctAnswer": 1,
  "explanation": "解説文",
  "tags": ["タグ1", "タグ2"],
  "reference": "参照元"
}
```

### ユーザー進捗

ユーザーの進捗データはブラウザのlocalStorageに保存されます。
キー: `poke-judge-progress`

## 注意事項

- このアプリは個人利用目的で作成されています
- 問題内容は公式ルールに基づいていますが、最新情報は公式サイトをご確認ください
- データはローカルに保存されるため、ブラウザのデータを削除すると進捗が失われます
